BuildForMac README

To build a TUI double-clickable application for MacOS X 10.3 or 10.4:

% python setup.py [--quiet] py2app

Notes:

  • Requires py2app, available from http://pythonmac.org/packages/.
  • For best results, install your own Python; see Python on the Macintosh for information. Using this python instead of the one that comes with MacOS X means that Python will be included in TUI, which in turns means it will be more likely compatible on more machines and versions of MacOS X.
  • If you build TUI on MacOS 10.4, it will also run on 10.3.9 (so long as all extra python libraries used by TUI were built properly). Binary installers for most of the required packages can be found at http://pythonmac.org/packages/. These are listed as for "MacOS X 10.3.9 and later"; the package names typically contain the string "macosx10.4" but they also work just fine with 10.3.9.
  • As of this writing 10.5 is not yet out and the compatibility issues are unknown. My best guess at this point is that that versions built under 10.4 will work under 10.5. However, it is also quite possible that newer versions built under 10.5 may not work under 10.3.9.
