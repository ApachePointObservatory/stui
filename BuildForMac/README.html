BuildForMac README

To build a TUI double-clickable application for MacOS X 10.3 or 10.4:

% python setup.py [--quiet] py2app

Notes:

  • Requires py2app, available from http://pythonmac.org/packages/.
  • If you build TUI on MacOS 10.4, it will only run on 10.3 if you used 3rd party python additions (such as numarray) that were built on MacOS X 10.3. Such packages can be found at http://pythonmac.org/packages/, listed as for "MacOS X 10.3+".
