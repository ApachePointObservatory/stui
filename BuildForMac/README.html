BuildForMac README

To build a TUI double-clickable application for MacOS X 10.3 or 10.4:

% python setup.py [--quiet] py2app

Requirements

The following python packages are required to build a Mac application (in addition to those packages required to run TUI)

  • easy_install.
  • py2app, available from PyPI (thus you can use easy_install to install it).
  • As of this writing, the released version of macholib is 1.2, which is too old. If building fails and you have macholib 1.2 then install the svn trunk of macholib as follows, running these commands from Terminal in a convenient directory:
    • svn co http://svn.pythonmac.org/macholib/macholib/trunk/ macholib-trunk
    • cd macholib-trunk
    • svn info # to find out the revision number
    • nano setup.py # change VERSION e.g. to 1.2.1-r<revision#>
    • python setup.py build
    • sudo python setup.py install

Notes

  • If your tuisdss uses packages that are not on the normal python path then supply a file tuisdss.pth in this directory that contains paths to all the extra python packages. The installer will add those paths to sys.path before building the application.
  • For best results, install your own Python; see Python on the Macintosh for information. Using this python instead of the one that comes with MacOS X means that Python will be included in TUI, which in turns means it will be more likely compatible on more machines and versions of MacOS X.
  • The resulting application will be compatible with whatever versions of MacOS X your Python and its packages are compatible with.
