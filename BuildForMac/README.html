BuildForMac README

To build a TUI double-clickable application for MacOS X 10.3 or 10.4:

% python setup.py [--quiet] py2app

Notes:

  • Requires py2app, available from http://pythonmac.org/packages/.
  • For best results, install your own Python 2.4; see Python on the Macintosh for information. Using this python instead of the one that comes with MacOS X means that Python will be included in TUI, which in turns means it will be more likely compatible on more machines and versions of MacOS X.
  • If you build TUI on MacOS 10.4, it will only run on 10.3 if all extra python libraries used by TUI (such as numarray) were built on or for MacOS X 10.3. Such packages can be found at http://pythonmac.org/packages/, listed as for "MacOS X 10.3+". If in doubt, build TUI on a machine running MacOS X 10.3; the resulting application will will run on both 10.3 and 10.4.
