BuildForMac README

Builds a STUI double-clickable application for MacOS X:

% python setup.py [--quiet] py2app

Requirements

The following python packages are required to build a Mac application (in addition to those packages required to run STUI)

  • setuptools.
  • py2app (which may be installed via pip or easy_install if you like).

Notes

  • For best results, install your own Python from python.org. Using this python instead of the one that comes with MacOS X means that Python will be included in STUI, which in turns means it will be more likely compatible on more machines and versions of MacOS X.
  • The resulting application will be compatible with whatever versions of MacOS X your Python and its packages are compatible with.
