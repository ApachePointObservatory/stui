TUI:Guiding:TSpec SlitViewer

The TSpec SlitViewer window controls the TripleSpec slitviewer. Most of the controls are standard guiding controls. A few tcam-specific controls appear in a box just above the status bar; those controls are described here.

  • Bkgnd Sub status and control of automatic background subtraction:
    • The first field shows the current state (Off or On).
    • Off turns off background subtraction.
    • On/New turns on background subtraction (if off) or takes a new background image (if on).
    • X cancels the current Off, On or New command. It only re-enables buttons, it does not send anything to the TripleSpec guider.
    • The final field shows the name of the background image. This field is only visible when background subtraction is on.
