TUI:Guiding:GCam

The GCam window controls the NA2 off-axis guider. Most of the controls are standard guiding controls. A few gcam-specific controls appear in a box just above the status bar; those controls are described here.

  • Focus the current focus is shown to the left. You may set a new focus in the field to the right.
  • Filter the current filter is shown to the left. You may select a new filter using the pop-up menu to the right.

  • Apply apply the requested focus and/or filter change.
  • Current restore the user-set focus and filter to the current settings.

Note: only the Apply button sends any commands to gcam. The other controls in this box specify what commands Apply should send.
