TUI:Guiding:Echelle Slitviewer

The Echelle Slitviewer window controls the echelle slitviewer. Most of the controls are standard guiding controls. A few ecam-specific controls appear in a box just above the status bar; those controls are described here.

  • Filter the current filter is shown to the left. You may select a new filter using the pop-up menu to the right.
  • Apply apply the requested filter change.
  • Current restore the user-set filter to the current setting.

Note: only the Apply button sends any commands to the slitviewer. The other controls in this box specify what commands Apply should send.
