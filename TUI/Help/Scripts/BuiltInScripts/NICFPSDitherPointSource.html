TUI:Scripts:Built In Scripts:NICFPS:Dither:Point Source

The NICFPS:Dither:Point Source script allows you to take a sequence of exposures in a square pattern plus a point in the middle. This pattern is typically used for point sources.

The requested number of exposures is taken at each point before moving on to the next. The order is center (your current position), upper left, upper right, lower right, lower left. Any point may be omitted by unchecking its checkbox before running the script.

Box size sets the size of one side of the square pattern, on the sky.

For example, if #Exp = 3 and Box Size = 20", the script does the following:

  • If Ctr is checked: take 3 exposures.
  • If UL is checked: move the boresight to (-10", +10"), relative to the initial boresight position, and take 3 exposures.
  • If UR is checked: move the boresight to (+10", +10") and take 3 exposures.
  • If LR is checked: move the boresight to (+10", -10") and take 3 exposures.
  • If LL is checked: move the boresight to (-10", -10") and take 3 exposures.
  • Restore the initial boresight position. This final offset occurs even if you cancel the script or the script fails.
