TUI:Scripts:Built In Scripts:NICFPS:Dither:Point Source

The NICFPS:Dither:Point Source script allows you to take a sequence of exposures in a square pattern plus a point in the middle. This pattern is typically used for point sources.

The sequence is to take #Exp exposures at each dither point before moving on to the next. The dither point order is shown as a number to the left of each checkbox.

Controls include:

  • A checkbox for each dither point. Uncheck the box (before the script starts) to skip the point.
  • Box Size: the size of one side of the square pattern, in arcseconds on the sky.

The input controls are ignored once the script starts (as usual for scripts).

As the script runs the status of each dither point is updated. This status is retained when the script ends so you can see what happened. It is cleared when you start a new run.
