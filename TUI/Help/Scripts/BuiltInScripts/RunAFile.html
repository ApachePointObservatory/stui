TUI:Scripts:Built In Scripts:Run A File

Run a file of commands. Each command runs to completion before the next command is executed. If any command fails, the script halts.

Each line of the file is one of:

  • A command, in the form: actor command
  • A blank line (ignored)
  • A comment starting with # (ignored)

Example file:

# sample file
tcc show status
tcc show time
