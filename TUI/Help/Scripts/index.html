STUI:Scripts

Information about writing user scripts and adding windows to STUI.

  • APO Documentation manuals for the instruments, hub and TCC.
  • Built In Scripts a description of the scripts built into STUI, including:
    • Run_Commands: run a sequence of commands (the simplest kind of scripting).
  • Scripting Tutorial a tutorial for writing scripts, with many examples.
  • Scripting Manual the scripting manual.
  • Programming how to add new windows to STUI and information about the internals of STUI (some of which may be of interest to script writers).
  • TUIAdditions where to put user scripts and code for new windows.
