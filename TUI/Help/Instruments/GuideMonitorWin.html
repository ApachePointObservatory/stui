STUI:Instruments:Guide Monitor Window

A strip chart showing various aspects of guiding, including:

Axis Offset

Axis corrections reported by the guider in arcseconds. These are the changes that are applied by the guider, or would be applied if corrections were enabled. All values are in arcseconds. The RA correction is distance on the sky (thus it has been scaled by cos(alt) to avoid blowing up near the pole). These values are reported by guider keyword axisChange.

Focus

Shows the actual secondary piston and the user-specified focus offset (which is the value the guider adjusts to maintain focus), both in µm. These values are reported by TCC keywords SecOrient (for piston) and SecFocus (for user offset).

Scale Factor

(plate scale/nominal scale) - 1, where larger scale (and hence larger scale factor) gives higher resolution, e.g. more pixels/arcsec. This comes from the TCC's ScaleFac keyword.
