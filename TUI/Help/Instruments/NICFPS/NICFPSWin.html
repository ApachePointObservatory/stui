TUI:Instruments:NICFPS:NICFPS window

Displays the current status of NICFPS spectrograph and allows you to change the configuration.

Warning: this is a preliminary implementation. Once the instrument builders have more time with the instrument we plan to refine the controls.

Pushing the "Show Config" button displays configuration controls. A "!" to left of a configuration control means it is not set to the current value. In other words, "!" indicates what Apply will change.

Hints: the contextual menu on each control and on the "!" indicator can return that control to the current NICFPS configuration.

The status and configuration items are as follows.

  • Filter. the current set of filters. NICFPS has 3 filters, but each common combination of filters has a name, which is shown here.
  • Etalon: shows whether the Etalon is In or Out of the beam. When the configuration control is set to "In", the following Etalon controls are visible:
    • Mode: etalon mode; one of Balance (the normal mode) or Operate (used for calibration).
    • RTime: response time of the etalon servo-feedback loop.
    • X Spacing: x position of the etalon.
    • Y Spacing: y position of the etalon.
    • Z Spacing: z spacing of the etalon. Z is along the beam, and thus most directly controls the wavelength.
  • Environment: the button controls whether to show or hide pressure and various temperatures. An OK to the right of the button means that the pressure and all temperatures are acceptable.

Below the items is a status bar that shows help and the state of the currently executing configuration command.

The buttons along the bottom are as follows:

  • Expose...: open the NICFPS's expose window.
  • Show/Hide Config: show or hide the congiguration controls.

    The following are only available if the configuration controls are showing:

  • Apply: apply the requested changes, if any.
  • Current: set all configuration controls to the current state of the instrument.
