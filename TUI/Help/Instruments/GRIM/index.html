TUI:Instruments:GRIM

Controls for GRIM, the near infrared GRIsm spectrometer and IMager, are distributed among two windows:

  • The GRIM window displays status and allows configuring GRIM. The status portion is always displayed. The configuration controls are hidden by default.
  • The GRIM Expose window controls and displays exposure sequences. Note that you can only take a Dark if GRIM is in Dark mode.

See also the GRIM documentation at APO.
