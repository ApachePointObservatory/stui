TUI:Telescope:Slew Window:Options:Keep Offsets

Allows you to specify which offsets should be retained and which offsets should be zeroed for a slew. Checked offsets are retained and unchecked offsets are zeroed. See Offset Types for more information on each type of offset.

  • Object: the object offset
  • Arc: the arc offset
  • Boresight: the boresight offset
  • Guide Corr: the guide correction
  • Calibration: the calibration offset

Warning: like all option panels, this one must be visible for its settings to have any effect. If you close (hide) it then default values will be used.

Defaults: restores this panel to its default values (the same values that will be used if you slew with this panel closed).
