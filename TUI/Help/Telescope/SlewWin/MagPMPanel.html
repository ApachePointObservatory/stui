STUI:Telescope:Slew Window:Options:Mag, PM

Allows you to specify the magnitude, proper motion, etc. of the object. The particular fields shown varies depending on the coordinate system. The full list of possibilities includes:

  • Mag: magnitude of object. Used only for display purposes.
  • Obs Date: date of observation; used for correcting proper motion and radial velocity. Warning: the date of observation is always the date of equinox; if you change one, you change the other. This is only an issue for FK5 and FK4 coordinates, the two coordinate systems that use a date of equinox.
  • PM RA/Long: proper motion in RA or Longitude, in "/cent. This is dRA/dt or dLong/dt, so for a given rate of motion across the sky the value gets larger nearer the pole.
  • PM Dec/Lat: proper motion in Dec or Latitude, in "/cent.
  • Parallax: parallax of object, in arcsec
  • Distance: distance to object, in parsecs; whether you see Distance or Parallax depends upon the coordinate system.

Warning: like all option panels, this one must be visible for its settings to have any effect. If you close (hide) it then default values will be used.

Defaults: restores this panel to its default values (the same values that will be used if you slew with this panel closed).
