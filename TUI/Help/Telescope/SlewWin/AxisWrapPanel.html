STUI:Telescope:Slew Window:Options:Axis Wrap

Allows you to specify a desired wrap for the azimuth axis and the instrument rotator. These requests have no effect unless you are in a region where more than one wrap is possible. If the axis is being driven in mount coordinates then your wrap preference will be recorded, but ignored (for mount coordinates you specify exactly the position you want; a different wrap is never chosen). The wrap preferences are:

  • Nearest pick the position nearest the current position
  • Negative pick the smaller angle
  • Positive pick the larger angle
  • Middle pick the position nearest the center of the allowed range of motion

Warning: like all option panels, this one must be visible for its settings to have any effect. If you close (hide) it then default values will be used.

Defaults: restores this panel to its default values (the same values that will be used if you slew with this panel closed).
