STUI:Telescope

This page provides information about the various telescope-related widgets, plus general topics such as user catalogs, coordinate systems and rotation types. STUI talks to the telescope via the TCC (Telescope Control Computer). Most of the telescope-related windows are available via the TCC menu of the main Status window.

TCC Windows (all are items in the TCC menu except for the Status window):

  • Fiducials Window
  • Focal Plane Window
  • Offset Window
  • Nudger Window for manual guiding
  • Secondary Focus Window
  • Sky Window
  • Slew Window
  • Status Window
