TUI:Tidbits:Standard Error

TUI prints certain messages to standard error. This information can be very useful when reporting bugs or debugging your own code. The location of standard error depends on your platform:

  • MacOS X: standard error goes to the console. Run /Applications/Utilities/Console to see the console.
  • unix: standard error is the terminal you used to launch TUI (unless you redirected it).
  • Windows: I have no idea. I suspect you have to run a special program to see standard error, but I have no idea what it might be.
