STUI SDSS 2.5m Telescope User Interface

This is the documentation for STUI, the user interface for the SDSS 2.5m telescope. The introduction will get you started. From within STUI you can jump to a topic by selecting Help in the contextual menu for a widget. Also, most windows have a status bar that shows brief help for any item the mouse is over.

  • Introduction a short guide to STUI.
  • STUI Menu items in the STUI menu
  • Telescope items in the TCC menu, plus help for user catalogs, offsets, coordinate systems and rotation types.
  • Instruments items in the Inst menu and instrument info, plus:
    • Guiding guiding.
  • Misc items in the Misc menu.
  • Scripts items in the Scripts menu, how to write scripts and even information on adding new windows to STUI.
  • Tidbits tips for entering sexagesimal numbers, where to find error logs and other tidbits.
