TUI Telescope User Interface

This is the documentation for TUI, the new telescope user interface for the APO 3.5m telescope. For basic information, see the introduction. For more detailed information you may follow the links below or, within TUI, jump directly to the appropriate place by using the Help entry in the contextual menu for a widget. To bring up the contextual menu, right-click, middle-click or (on a Mac with a one-button mouse) control-click on the widget. Also, most windows have a status bar that shows brief help for any item the mouse is over.
