TUI:Misc:Truss Lamps window

Control the truss lamps: calibration lamps mounted on the secondary cage. Press a button to immediately toggle the state of the associated lamp. The control background color will change to pink (or the "Bad Background" preference) until the lamp has changed state.

Warning: as of 2005-01 there are still multiple ways to control the truss lamps, and they do not communicate with each other. So if you want to use TUI to monitor or control the truss lamps, make sure that all users control the lamps via TUI! Otherwise:

  • The displayed lamp state will be wrong.
  • Non-TUI lamp commands block TUI access for a period of time (something has to time out before TUI has control again, and even then, the displayed state will be wrong).

This is PR 63 and should be fixed at APO some day.

Notes:

  • If the lamp controls say "Rebooting" then you should wait until the tlamps controller finishes rebooting.
  • If a lamp control says "Unknown", you can still try to toggle the lamp, but it is likely that the lamp controller needs some help.
