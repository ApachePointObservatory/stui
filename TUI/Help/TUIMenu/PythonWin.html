STUI:STUI Menu:PythonWin

The Python window allows you to execute arbitrary Python code. This may be helpful for executing short scripts, as well as more intensive programming such as writing or debugging new widgets. More information about programming STUI may be found here.

The Python window is brought up by selecting Python from the STUI Menu. The entities are as follows:

  • The main text area is where you enter Python code.
  • Open...: open a text file (presumably containing Python code) in its own Python window.
  • Save...: save the current text to a file.
  • Clear: clear the text area. (This button may go away if I ever implement cmd-A for select all).
  • Run: run the current text.
