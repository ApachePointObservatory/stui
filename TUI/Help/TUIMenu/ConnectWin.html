TUI:TUI Menu:Connect Window

The connect window allows you to connect to "hub" at the observatory. It is brought up by selecting Connect from the TUI Menu. The entities are as follows:

  • User Name: typically your last name, but any descriptive word is fine. This is the name by which you will be seen by others. The hub will alter it a bit if necessary to avoid duplication.
  • Program Name: the name of your program as assigned to you by APO. Do not improvise! Passwords and permissions are both associated with program numbers.
  • Password: the password associated with the program name.

  • Status bar: displays the current status of the connection.

  • Connect button: initiates the connection, Be sure the program name and password are entered first.
  • Cancel button: cancels the pending connection, if any. It does not close the window. Furthermore, closing the window has no effect on a pending connection or disconnection.
