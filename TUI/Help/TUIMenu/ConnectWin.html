TUI:TUI Menu:ConnectWin

The connect window allows you to connect to "hub" at the observatory. It is brought up by selecting Connect from the TUI Menu. The entities are as follows:

  • User Name: typically your last name, but any descriptive word is fine. This is how you will show up to other users (though if there are duplicates, the hub will alter you choice a bit, probably by adding a digit).
  • Program Name: a name assigned by APO with an associated password.
  • Password: the password associated with the program name.
  • Status: displays the current status of the connection.
  • Connect button: initiates the connection, Be sure the program name and password are entered first.
  • Cancel: closes the dialog box without connecting.
