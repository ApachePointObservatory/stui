Notes

This python is newer than RedHat's and includes support for Tkinter and the additional packages listed below.

To use it, you must specify the whole path /net/python/bin/python or update your PATH to include /net/python/bin (see next). Otherwise you will get RedHat's Python.

If you use tcsh and Chance Reschke's standard ENV setup, then you can update your path by adding the following to ~/ENV/.cshrc.personal: 

setenv PATH "/net/python/bin/:"$PATH

Additonal packages installed

All packages are installed in the standard location, so you can simply import them without having to define the environment variable PYTHONPATH. That location is /net/python/lib/python2.3/site-packages/.

  • Numeric-23.1: efficient matrix manipulation. See the on-line manual and home site. Installed 2004-01-07.
  • tkSnack 2.2.3: a sound library based on tk. See Tcl Additions for more information. Installed 2004-01-08.

Installation Notes

Unless noted below, all software was built according to the included instructions and those instructions contained no surprises. For Python packages, the typical installation is: 

/net/python/bin/python setup.py install

Python

Python installation used the configure flag --enable-unicode=ucs4 because of a note in the Python README to the effect that RedHat's tcl/tk uses ucs4 instead of the ucs2 that Python expects. Presumably at some point this flag will no longer be needed. The exact sequence was: 

./configure --prefix=/net/tui --enable-unicode=ucs4
make
make test
make install
